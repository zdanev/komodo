# KoMoDo

Set of command line utilities for common internet services.

Drop me a line to get involved!

## Examples

### Weather

```cmd
weather
weather 92656
```

### Stocks

```cmd
stocks
stocks AAPL MSFT GOOG
```

### CNN

```cmd
cnn
cnn world
cnn us
cnn travel
...
```

### RSS news

```cmd
news
```

### Google/Bing

```cmd
google komodo

```
